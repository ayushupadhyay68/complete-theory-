	displaying quotes in the web page -:

		
		<q></q> - is used to display the short quotes in the web page by applying double quotes in the start and end of the wrapped content.
		
		<blockquote></blockquote> - is used to display the long quotes in the web page.
		
	
	
	<abbr></abbr> - is used to display the abbereviations in the web page.
	
		attributes -:
		
			title - it is a global attribute which contains the title of the element which displays when the user mouse over on that element.
			
	

	character entities -: in html some characters are reserved so we can not use them or characters which are not available on the keyboard for those we use character entites.
		
			syntax -:
			
				&entityname;
				
				or
				
				&#entitycode;

	creating list in the web page -:
	
		types of list -:
		
			i. Ordered List - items display in the order like in numbers or alphabetical order.
			
			ii. unordered list - items display in the bullets.
			
			iii. definition list - it is used to display the description of the item.
			
			
		a. creating ordered list -
		
			<ol></ol> - is used to create the ordered list.
			
			<li></li> - is used to define / provide the list item for the ordered list.
			
		b. unordered list -
		
			<ul></ul> - is used to create the unordered list.
			
			<li></li> - is used to create the list item for the unordered list.
			
		
		c. definition list -
		
			<dl></dl> - is used to create the definition list.
			
			<dt></dt> - is used to define the term / item for the defintion list.
			
			<dd></dd> - is used to provide the definition description / data definition for the definition list.
			
	
		changing the type of the ordered or unordered list -:
		
			type - is used to change the list type to the given value.
			
				value -: a | A | I | i | 1 (for ordered list) - disk | circle | square (for unordered list)
